# Vrinda-store-sales-dashboard
## Overview
An end-to-end Excel data analysis project for Vrinda Store, covering data cleaning, pivot tables, and an interactive dashboard with slicers to analyze sales performance.

## Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Data Cleaning, Dashboard Design)

## What's Inside
- **Sales vs Order** — comparison of sales volume against order counts
- **Vrinda Store Report 2022** — full year performance summary
- **Men vs Women** — sales breakdown by gender
- **Order Status** — tracking of order fulfillment status
- **Top 5 States** — highest performing states by sales
- **Age & Gender** — customer demographic analysis
- **Channels** — sales breakdown by channel (online/offline/marketplace)

## Key Skills Demonstrated
- Data cleaning and preparation
- Pivot table design and analysis
- Interactive dashboard building with slicers
- Translating raw sales data into business insights
